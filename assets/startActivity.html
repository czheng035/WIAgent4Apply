Explanation

It is simple to invoke an action:

  1. Instantiate an Intent object with an action, data type and the data you want to send
  2. Call window.naviagator.startActivity with the intente object and option success and failure callbacks

Code

var intent = new Intent("http://webintents.org/share", "text/uri-list", location.href);

window.navigator.startActivity(intent, function(data) { /* on success */ });
